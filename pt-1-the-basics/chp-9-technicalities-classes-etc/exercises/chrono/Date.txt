Date Class
==========

Description:
------------
 * Allows the creation of a valid date instance
 * Allow created date instance to be displayed in different formats
 * Allow access to date instance (non-modifying) members,
   - date()
   - month() -- e.g. Feb or 2
   - year()
   - day_of_week()
 * Enable date to be updated,
   - add_day()
   - add_month()
   - add_year()
 * Additional functions
   - next_sunday()
   - next_weekday()
   - next_workday()
